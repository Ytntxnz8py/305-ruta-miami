# 305 Ruta Miami

Directorio bilingüe (ES/EN) de experiencias en Miami, con foco en el nicho náutico y de pesca. Monetizado con listings de negocios pagados (suscripción mensual B2B).

🌐 **Sitio en vivo:** [305rutamiami.com](https://305rutamiami.com/)

## Stack

HTML5 + CSS3 + JavaScript ES5 puro (sin frameworks, sin build) · GitHub Pages · Supabase (backend) · GA4 · Brevo (email) · Formspree (formularios B2B).
