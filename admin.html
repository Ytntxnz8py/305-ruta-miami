Activar Google Analytics

⚪ No configurado

Para ver datos reales de visitas, conecta tu propiedad GA4 mediante el proxy.

  1. 1
    Despliega el Google Apps Script proxy

    Sigue las instrucciones en _privado/SETUP_GAS.md. El proxy actúa como puente entre el sitio estático y la GA4 Data API.

  2. 2
    Copia la URL del Web App desplegado

    Formato: https://script.google.com/macros/s/AKfyc.../exec

  3. 3
    Configura la URL del proxy

    Pega la URL en assets/js/admin-config.js → campo GAS_PROXY_URL. Haz commit y push.

    GAS_PROXY_URL: 'https://script.google.com/macros/s/AKfyc.../exec'
  4. 4
    Verifica en el Dashboard

    El tab Dashboard mostrará "GA4 Real" en lugar de "LocalStorage" cuando el proxy esté activo.
